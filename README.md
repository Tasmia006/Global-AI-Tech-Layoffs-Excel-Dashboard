# Global-AI-Tech-Layoffs-Excel-Dashboard
I built this interactive dashboard to analyze tech layoff trends between 2020 and 2026, looking at which industries were hit the hardest and how AI is changing the job market.

- Excel
- Pivot Tables

## Key Insights
- Most impacted countries
- Layoff trends by year
- Industry comparison
- Stock market reaction

This project analyzes global tech layoffs using Excel dashboards.
